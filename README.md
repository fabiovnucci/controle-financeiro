# Controle Financeiro

Projeto de **Controle Financeiro** desenvolvido com **HTML, CSS e JavaScript básico**, com foco em aprendizado e boas práticas de front-end.

O usuário pode:

- Adicionar transações de **ganho** ou **gasto**  
- Visualizar **lista de transações**  
- Ver **totais de ganho, gasto e lucro**  
- Receber **alerta de valores inválidos** (como números negativos ou zero)

## Tecnologias utilizadas

- HTML  
- CSS (Tailwind ou Bootstrap opcional)  
- JavaScript (arrays, funções, manipulação de DOM, alertas)  

## Objetivo do projeto

Este projeto tem como objetivo **praticar front-end básico**, manipulação de DOM e regras de validação. É um exemplo de como criar uma aplicação simples que interage com o usuário de forma dinâmica.

## Como usar

1. Abra o arquivo `index.html` no navegador  
2. Digite um valor e selecione o tipo de transação  
3. Clique em "Adicionar"  
4. Veja a lista de transações e os totais atualizados
